Principais formatações

Negrito/Destaque

Para usar algo em negrito basta utilizar a tag <b> (sem semântica)
Ou utilizar a tag <strong> para textos em negrito (com semântica)

Itálico/Ênfase

Para usar um texto em itálico basta utilizar a tag <i> (sem semântica)
Para usar um texto em itálico com enfase (sentido semantico), basta utilizar a tag <em> (com semântica)

Marcador de Texto

Para marcar um texto usamos a tag <MARK>

Podemos também usar um texto marcado em colorido

Texto grande e texto pequeno

Para deixar um texto grande podemos utilizar a tag <big>. No entanto, o <big> não deve mais ser utilziado pois se tornou obsoleto!
O ideal é alterar o tamanho da letra no CSS

Para deixar um texto pequeno podemos utilziar a tag <small>

Texto deletado

Para riscar um texto deletado utilizamos a tag <del>

Texto inserido

Para marcar um texto inserido basta utilizar <ins>

Texto sobrescrito

Para marcar um texto sobrescrito utilizamos <sup>

Texto subscrito

Para marcar um texto subscrito utilizamos <sub<
